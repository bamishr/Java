## 代理模式

### 概念
为其他对象提供一种代理，以控制对这个对象的访问。代理对象起到中介的作用，可以去掉功能服务
或增加额外的服务。

例如：火车票代售处是火车站的代理，它没有退票服务，但是有电话预约等服务。

### 分类

#### 远程代理
为不同地理位置的对象，提供局域网代表对象。例如远程监控。

#### 虚拟代理
根据需要，将一个资源消耗很大的对象进行延迟，真正需要的时候才创建。例如：网页渲染时，使用
虚拟的图片代替真实的图片，当真实图片加载完成后才显示。

#### 保护代理
控制对一个对象的访问权限。例如权限控制。

#### 智能代理
提供对目标对象额外的服务。
